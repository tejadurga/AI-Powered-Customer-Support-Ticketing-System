# AI-Powered Customer Support Ticketing System

## Project Overview

The AI-Powered Customer Support Ticketing System is a Salesforce-based solution designed to streamline customer support operations through intelligent case management, SLA tracking, AI-assisted responses, and reporting.

This project leverages Salesforce's powerful platform capabilities, including Agentforce, Prompt Builder, Flows, Validation Rules, Reports, and Dashboards, to improve support efficiency and enhance customer experience.

---

## Features

### Case Management
- Create, update, and manage customer support cases.
- Track case status and priority levels.
- Assign cases to support agents.

### SLA Tracking
- Monitor SLA Due Dates.
- Track resolution timelines.
- Improve customer service compliance.

### AI Suggested Responses
- Generate intelligent customer support replies using Prompt Builder.
- AI-assisted response suggestions for faster issue resolution.
- Agent review before sending responses.

### Agentforce Integration
- AI-powered support assistant.
- Case-based response generation.
- Enhanced agent productivity.

### Reporting & Analytics
- Open Cases by Priority Report.
- Agent Performance Report.
- Case workload tracking and performance monitoring.

### Security & Access Control
- Support Agent Profile.
- Support Manager Profile.
- Permission Sets for AI and Reporting access.

## System Workflow

1. Customer support case is created.
2. Case details are stored in Salesforce.
3. Agent reviews the case.
4. Agentforce analyzes case information.
5. AI generates a suggested response.
6. Agent reviews and sends the response.
7. Case resolution metrics are tracked through reports.
